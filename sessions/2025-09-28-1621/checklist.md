# Session Checklist

**Before**

- [ ] Load `principles.md` and `claude.md`
- [ ] Define goal, success criteria, and tests
- [ ] Open feature branch

**During**

- [ ] Plan → act → validate in small steps
- [ ] Keep edits minimal; review diffs
- [ ] Update tests/docs

**After**

- [ ] Summarize changes and rationale
- [ ] Record Coach Notes in `learning_log.md`
- [ ] Open PR with validation evidence
